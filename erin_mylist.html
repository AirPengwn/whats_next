☁ Set up cloud sync — see your list on any browser
Your saved items are stored in a free JSONBin.io account. Once set up, your list syncs across every browser and device automatically. Takes about 2 minutes.
  1. Go to jsonbin.io and create a free account
  2. In your dashboard, create a new Bin with this content: {"items":{}}
  3. Copy the Bin ID (looks like: 64a3f…)
  4. Go to Account → API Keys, create a key, copy it
  5. Paste both below and click Save

🔒 Your API key is stored only in this browser's localStorage. It is never sent anywhere except JSONBin.io's API.
